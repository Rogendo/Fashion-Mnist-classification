# Fashion-Mnist-classsification 

This repository contains fashion mnist notebook. It classifies the types of clothing into 10 classes. 

These classes are: 
    'T-shirt/Top',
    'Trouser',
    'Pullover',
    'Dress',
    'Coat',
    'Sandal',
    'Shirt',
    'Sneaker',
    'Bag',
    'Ankle boot'
